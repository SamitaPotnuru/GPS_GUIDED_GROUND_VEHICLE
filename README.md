# GPS Guided Ground Vehicle

## Overview
This project focuses on designing and developing a **GPS Guided Ground Vehicle** that can navigate to specific coordinates using GPS data. The system uses an Arduino-based controller along with sensors and motor drivers to guide the vehicle automatically.

## Objective
To build an autonomous ground vehicle that can move towards a target location using GPS navigation and sensor-based direction control.

## Hardware Used
- Arduino Mega 2560  
- Ublox NEO-6M GPS Module  
- HMC5883L Compass Module  
- L293D Motor Driver  
- HC-05 Bluetooth Module  
- 11.1V Li-Po Battery  
- DC Motors and Wheels  
- Jumper Wires

## Software Used
- Arduino IDE  
- MIT App Inventor (for mobile control)

## Working Principle
The GPS module receives location coordinates from satellites. The Arduino processes this data along with compass readings to determine direction. Based on the calculated path, the motor driver controls the motors to move the vehicle toward the target location.

## Features
- GPS-based navigation  
- Direction control using compass sensor  
- Arduino-based control system  
- Wireless communication using Bluetooth

## Applications
- Autonomous delivery vehicles  
- Agricultural field monitoring  
- Search and rescue robots  
- Military and surveillance vehicles

## Future Improvements
- Obstacle detection using ultrasonic sensors  
- Improved path planning algorithms  
- Real-time tracking using IoT
