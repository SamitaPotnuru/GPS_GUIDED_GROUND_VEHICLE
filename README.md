# GPS Guided Ground Vehicle

Arduino based autonomous ground vehicle using GPS navigation,
magnetometer compass, ultrasonic collision avoidance and Bluetooth control.

## Hardware
- Arduino Mega 2560
- L293D Motor Shield
- HMC5883L Compass
- Ublox Neo 6M GPS
- HC05 Bluetooth
- Sensor

## Features
- GPS waypoint navigation
- Bluetooth manual control
- Compass based heading correction
- Collision avoidance
